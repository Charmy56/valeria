basic html tags

This are the commonly used in HTML. They are usually enclosed in carets "<__>"

  1. html.../html - Declares the webpage to be written in HTMl
  2. title.../title - Defines the title to be displayed on the page
  3. Head.../head - delimits the pages head
  4. h1.../h1 - Delimits the page's head Usually from h1...h6
  5. hr.../hr - Inserts a horizontal rule
  6. ul.../ul - Brackets an unordered list
  7. ol.../ol - Brackets a numbered list
  8. body.../body - Delimits the page's body
  9. li.../li - Brackets an item in an ordered or numbered list
  10. a href="url"resource.../a - Defines a hyperlink
  11. image src="address" aH="" - Displays an image here
  12. p.../p - Starts a paragraph
  13. u.../u - underlines a heading in a page
